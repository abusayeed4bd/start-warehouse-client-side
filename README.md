# Simple warehouse website with react js, mongodb and express

Live Link [https://react-node-mongo-warehouse.web.app/](https://react-node-mongo-warehouse.web.app/).

## Technology

### front end
- react js
- bootstrap
- firebase authentication
- react toastify
- react BrowserRouter
- custom hooks

### Backend
- node js
- express js
- mongodb
- heroku
